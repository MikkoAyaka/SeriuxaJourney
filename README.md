## 背景故事

一场名为"幽匿"的神秘感染犹如暴风骤雨，席卷了这片曾经充满生机的土地。灾难之后，地面已经被幽匿毒雾和死亡的阴影所笼罩，无法再为人类提供庇护。然而，幸存者们，这些从深渊边缘挣扎逃离的勇士们，经过无数痛苦和困难，凭借着顽强的意志和无尽的创新，建立起了悬浮在天空中的避难城市——Siriuxa。

在这座空中堡垒中，人们找到了一丝安宁和一线希望。然而，为了维持这个人类的最后堡垒，他们必须冒险下到地面，去寻找生存所需的物资，去面对那些隐藏在黑暗中的未知威胁。在这个充满挑战和未知的世界中，每一个决定，每一次探索，都可能改变他们的命运。在这片被感染的土地上，他们必须找到生存的希望，才能重塑这个世界的未来。

## 玩法内容

### 任务系统

任务系统是我们游戏的核心玩法。目前，我们的任务类型只有可组合任务(ComposableTask)。这种任务可以在运行时修改其任务修饰、任务环境、任务目标，以动态改变任务玩法。目前，我们已经可以组合出50+种类的任务玩法，并且种类数量仍在增加。

玩家完成任务可以获得物资、特殊道具、经济奖励、经验奖励等。同时，任务支持单人游玩和组队游玩两种方式。

可组合任务的具体玩法全然由任务修饰、任务环境、任务目标决定。例如，在某个可组合任务中的搭配可能如下：

- 任务修饰：幽匿感染、安全作业、时间陷阱、AI强化
- 任务环境：地狱
- 任务目标：生存

在这个任务中，玩家将在地狱进行任务，需要尽可能生存到最后一刻再撤离。在任务过程中主要会受到任务修饰效果影响。具体的修饰效果请参见后续介绍。

| **任务目标类型** | **目标介绍**                                            |
|------------|-----------------------------------------------------|
| 生存         | 玩家只需要尽可能坚持到最后一刻再撤离，即可完成任务。任务报酬丰富程度与玩家生存得分相关。        |

### 难度系统

所有任务都可以在开始前选择合适的难度。难度会影响到如任务成本、任务报酬倍率、受伤倍率等具体任务属性，根据不同的任务有不同的影响效果。目前共有：轻松、常规、困难、专家，四种难度可供选择。在未来，我们可能会增加更多种类的难度。

### 队伍系统

玩家可以自由组队，队伍内拥有独立聊天频道，可以查看队内成员在线情况，例如当前处于哪个世界，是否正在进行任务等。如果要开始进行多人任务，由队长选择任务模式和任务难度，等待队伍内所有成员到达指定位置准备完毕后，即可开始进行多人任务。

### 阶段系统

不同类型的任务拥有不同的阶段。例如，可组合型任务拥有对应的等待阶段、准备阶段、游戏阶段、结束阶段共四个阶段。在未来的剧情任务中可能拥有更多游戏阶段以及阶段分支，不再是简单且固定的4个线性游戏阶段。

### 特殊物品系统

我们的游戏中有一些特殊的物品，它们与原版物品不同，往往拥有独一无二的主动/被动效果。在某些情况下，这些物品可以被携带到任务中。

### 怪物生成系统

我们的怪物生成系统与原版的不同，在任务中完全按照插件原创的生成算法进行怪物生成，不采用原版怪物生成。插件算法与原版算法的对比如下：


|                            | Minecraft 原版怪物生成算法 | Siriuxa 原创怪物生成算法 |
| -------------------------- | -------------------------- | ------------------------ |
| 怪物生成密度               | 密度较低，较为分散         | 密度较高，较为集中       |
| 海上生成怪物               | ❌                         | ✅                       |
| 狭窄空间生成怪物           | ❌                         | ✅                       |
| 玩家抱团额外刷怪           | ❌                         | ✅                       |
| 严格控制怪物数量           | ❌                         | ✅                       |
| 异步计算                   | ❌                         | ✅                       |
| 自定义群系怪物种类         | ❌                         | ✅                       |
| 自定义不同类型怪物生成概率 | ❌                         | ✅                       |

### 记录系统

我们的游戏有强大的记录系统。每当玩家完成一次任务，任务中的详细数据都会被记录到数据库中。与此同时，玩家的背包信息也会随着每次任务的开始与结束保存到服务器自动备份。
